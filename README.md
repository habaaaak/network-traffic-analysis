# Network Traffic Analysis

This project automates network diagnostics and analyzes network traffic using Python, Bash, and packet capture tools.

## Features
- Automated ping and traceroute using Python and Bash
- Packet capture using tcpdump
- Network traffic analysis using Wireshark (.pcap files)

## Tools Used
- Python
- Bash
- tcpdump
- Wireshark
- Linux

## Purpose
This project demonstrates hands-on experience with network protocols, traffic analysis, and system-level diagnostics.

## Results

- Successfully automated network diagnostics and packet capture using Python and Bash  
- Generated `.pcap` files for in-depth analysis in Wireshark and `.txt` logs for readable output  
- Captured and analyzed ICMP traffic to observe packet flow and network behavior  
- Demonstrated the ability to identify network paths, latency, and communication patterns  
- Simulated a real-world network troubleshooting and analysis workflow

## Key Takeaways

- Gained hands-on experience with network-layer protocols (ICMP)  
- Improved understanding of how packets travel across networks  
- Learned how to automate system-level tasks using scripting  
- Strengthened debugging and network analysis skills
