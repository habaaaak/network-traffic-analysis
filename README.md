# Network Traffic Analysis

This project automates network diagnostics and analyzes network traffic using Python, Bash, and packet capture tools.

## Features
- Automated ping and traceroute using Python and Bash
- Packet capture using tcpdump
- Network traffic analysis using Wireshark (.pcap files)

## Tools Used
- Python
- Bash
- tcpdump
- Wireshark
- Linux

## Purpose
This project demonstrates hands-on experience with network protocols, traffic analysis, and system-level diagnostics.
